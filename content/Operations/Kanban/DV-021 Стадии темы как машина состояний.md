## Цель
В FAQ описан жизненный цикл темы (предложена → созревает → назначена → обсуждается → синтез готов), но в коде это просто строка. Нужны валидные переходы и аудит.

## Definition of Done
- [ ] Таблица `topics` имеет поле `stage` enum: `proposed | maturing | scheduled | discussing | synthesized | published | archived`
- [ ] Создана таблица `topic_stage_history` для аудита переходов
- [ ] В коде определены разрешённые переходы (например, нельзя из `proposed` сразу в `synthesized`)
- [ ] API `PATCH /api/topics/:id` валидирует переход и пишет историю
- [ ] UI отображает текущую стадию и доступные действия (кнопки только релевантных переходов)
