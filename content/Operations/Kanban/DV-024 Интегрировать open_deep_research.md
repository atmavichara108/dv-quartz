## Цель
При подготовке темы дать возможность запустить агентский ресёрч: автоматический сбор источников, краткие выжимки, противоречия. Сэкономить часы кураторской работы.

## Definition of Done
- [ ] Сделан tech-spike: open_deep_research поднят локально в Docker
- [ ] Подключён mcp-searxng как поисковый бэкенд (приватный поиск)
- [ ] На странице темы появляется кнопка «Углубить тему» (только для admin/member-researcher)
- [ ] После запуска — фоновая задача, по завершении в материалах появляются собранные источники с пометкой `auto-researched`
- [ ] Есть механизм лимита по стоимости (не более N запросов / месяц)
- [ ] Документировано в ADR-006
