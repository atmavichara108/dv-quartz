# Сеть ячеек ДВ

## Принцип
Каждая ячейка автономна. Сеть — не иерархия, а экосистема.
Ячейки связаны через общие ресурсы, общую методологию и общую базу знаний (DV Hub).

## Типы связей между ячейками
- **Тематическая** — две ячейки работают над одной темой с разных сторон
- **Ресурсная** — одна ячейка помогает другой (эксперт, инструмент, контакт)
- **Контентная** — совместное производство (дебаты между ячейками)
